# Student Attendance Management System (SAMS)
## DBMS Mini Project

**Project Title:** Student Attendance Management System  
**Database:** MySQL

### Features
- Manage Students, Teachers & Classes
- Daily Attendance Marking (Present/Absent/Late/Excused)
- Attendance Reports & Percentage Calculation
- Low Attendance Alerts
- Basic Leave Management

### Setup Steps
1. Create Database: `CREATE DATABASE sams;`
2. Run `schema.sql`
3. (Optional) Run `sample_data.sql`
4. Use queries from `queries.sql`
